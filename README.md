## 🗂️ What I did · 맡았던 일

- **실시간 대시보드 WebSocket** — 중복 조회를 '보는 사람' 단위 1회 조회로 전환, DB 호출·카운트 어긋남 동시 해결
- **DBMS 메타 트리 API** — 여러 번 부르던 API를 트리 한 번으로 통합, '그래프 가능' 판단을 백엔드로 일원화
- **센서 측정값 수집** — 현장 센서값을 AWS IoT Core/MQTT로 받아 변환·저장
- **그 외** — 인증 서버 분리, 사내 공용 로깅 라이브러리

## 💡 How I work · 일하는 방식

- **작은 값도 검증** — 미세한 수치 오차도 원인까지 추적
- **점진적으로 바꾸기** — 신·구 방식을 함께 돌린 뒤 안전하게 전환
- **피드백을 새 시도로** — 사내 사용 팀의 불편을 듣고 문서 뷰어 개선을 직접 기획·PR
