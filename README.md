## 🗂️ What I did · 맡았던 일

- **실시간 대시보드 WebSocket** — 구독자별 중복 조회를 1회 조회 후 fan-out 구조로 전환, DB 호출↓·카운트 정합성 확보
- **핵심 도메인 (비콘·장소·문서)** — 비콘 생애주기·장소 계층 CRUD·문서 관리를 구현
- **DBMS 메타 트리 API** — 여러 번 부르던 API를 트리 단일 호출로 통합, 항목별 차트 표시 가능 여부 판단을 백엔드로 일원화
- **인증·계정 관리 서버 재구현** — 기존 서버를 NestJS로 재구현. 부서·초대·문의는 주도, 유저·프로젝트는 공동 개발
- **그 외** — 사내 공용 로깅 라이브러리, NestJS 온보딩

## 💡 How I work · 일하는 방식

- **작은 값도 검증** — 미세한 수치 오차도 원인까지 추적
- **점진적으로 바꾸기** — 신·구 방식을 함께 돌린 뒤 안전하게 전환
- **피드백을 새 시도로** — 사내 팀이 겪던 불편에서 출발해, 기획을 경험해보고 싶어 팀에 양해를 구한 뒤 문서 뷰어 개선을 기획 단계부터 맡음. 권한·실패 경로를 **동작 프로토타입**으로 명세해 합의 후 BE 구현
